

CLOSE: A toolkit for **C**NA/**LO**H analysis with **Se**quencing data
====

xuefeng.wang@stonybrook.edu 

hxchen@ucdavis.edu


CLOSE is a toolkit for **C**NA and **LO**H analysis with **SE**quencing data. Current pipeline majorly facilitates the analysis  on paired tumor and normal samples. It provides flexible entry points for different users: it accepts BAM files, VCF files or intial segmention files as input.
